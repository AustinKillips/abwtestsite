# All Bikes Welcome Static Starter

This is a stripped down HTML/CSS rebuild of the public site structure at `https://allbikeswelcome.org/`.

## Files

- `index.html`: editable page content and section order
- `styles.css`: brand variables, layout, responsive rules, and font loading
- `assets/`: copied SVG logos, arrows, and shapes with web-friendly names
- `fonts/`: copied Crimson Pro, Libertario, and Respect font files

## Palette

The CSS variables are pulled from the SVG asset colors:

- `--cream: #f1daaf`
- `--ink: #271912`
- `--coral: #ff6455`
- `--blue: #3e8fc6`
- `--pink: #d4809d`
- `--brown: #a7582a`
- `--green: #2fbe6b`
- `--yellow: #ffe738`

Swap these in `:root` to try new color collections quickly.

## Editing Notes

The newsletter form is visual only. Replace the `form` action or connect it to your provider when you are ready.

The impact numbers are placeholders so the layout has something to hold. Replace them with your current report numbers.
